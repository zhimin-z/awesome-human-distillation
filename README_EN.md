## What is Human Distillation?

Take a real person who existed in your life — a colleague, a mentor, an ex, a parent — and distill them into a callable [Claude Code Agent Skill](https://code.claude.com/docs/en/skills) through chat logs, writing, voice, or any other trace they left behind.

## Contributing

**Submit an existing Skill**

Found a Human Distillation Skill? [Open an Issue](https://github.com/mliu98/awesome-human-distillation/issues/new?template=add-skill.yml) — no GitHub experience required.

**Build your own**

1. Reference the structure of [colleague-skill](https://github.com/titanwings/colleague-skill)
2. Collect distillation materials (chat logs, essays, recordings, etc.)
3. Write it as a Claude Code Agent Skill
4. Publish to GitHub and submit to this list
